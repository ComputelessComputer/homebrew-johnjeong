# homebrew-johnjeong

Homebrew tap for `johnjeong` formulae.

## Install

```sh
brew tap ComputelessComputer/johnjeong
brew install ComputelessComputer/johnjeong/johnjeong
```

## Run

```sh
johnjeong
```

To point at a custom content directory:

```sh
JOHNJEONG_CONTENT_DIR=/path/to/part-of-my-brain johnjeong

If content is missing, it will fetch from GitHub (requires `git`).
```
